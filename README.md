PsyBridge MBTI Assessment

A modern web application for MBTI personality assessment: users provide demographic data and complete a 60-question survey to instantly receive their MBTI type and a downloadable report.
The app features a React/Vite frontend, FastAPI backend, and machine-learning MBTI prediction.

Features
- Intuitive, mobile-friendly React UI
- 60-question Likert scale MBTI questionnaire
- User demographic collection (name, age, gender)
- Instant MBTI prediction using trained ML model
- Downloadable CSV report for patient and quiz results
- Robust handling of browser refreshes and partial completion
- Persistent backend with FastAPI + scikit-learn

Tech Stack
- Frontend: React, Vite, HTML/CSS/JavaScript
- Backend: FastAPI, Python, scikit-learn
- Deployment: Vercel (frontend), Railway (backend)
- State Management: React hooks + sessionStorage
